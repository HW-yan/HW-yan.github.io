必做題：TJO 98

題目：
光速 c = 299792458 m/s
一年有365天，一周有7天，一天有24小時，一小時有60分鐘，一分鐘有60秒。

  • 光秒 Light-second(LS)
  • 光分 Light-minute(LM)
  • 光時 Light-hour(LH)
  • 光日 Light-day(LD)
  • 光週 Light-week(LW)
  • 光年 Light-year(LY)

範例輸出
1 Light-second(LS) is 299792458 metres.
1 Light-minute(LM) is XXXXXXXXX metres.
1 Light-hour(LH) is XXXXXXXXX metres.
1 Light-day(LD) is XXXXXXXXX metres.
1 Light-week(LW) is XXXXXXXXX metres.
1 Light-year(LY) is XXXXXXXXX metres.

解題誤區：

㊙️ (錯誤)
Light-year(LY) = Light-week(LW) X 356

㊙️ (正確)
Light-year(LY) = Light-week(LW) X (356/7) = Light-day(LD) X 356

🎌必做題：TJO 536

https://toj.tfcis.org/oj/pro/536/

題目：
給四個座標x0, x1, x2, x3，判斷是否會重疊
注意：$$x0<=x1 且 x2<=x3$$

解法I：必錯

解題誤區：

㊙️ 一開始沒想法，所以直接拿範例硬幹邏輯，當然錯。

解法II：正向窮舉 (列出所有Yes的情況)

解題誤區：

㊙️ 要冷靜下來列，不然容易錯很多次
![[【課程｜SCIST演算法】Week1 下午場：作題記錄 2022-10-04 02.51.18.excalidraw]]
狀況1總結：

  • 1️⃣ x2< x0 < x3 < x1
  • 2️⃣ x0 < x2 < x1 < x3
  • 3️⃣ x0 < x2 < x3 < x1
  • 4️⃣ x2 < x0 < x1 < x3
    ▶️ x2 < x1 < x3 4️⃣ ｜ 8️⃣ 5️⃣
    ▶️ x0 < x2 < x1 2️⃣ 3️⃣ ｜7️⃣
    ▶️ x0 < x3 < x1 3️⃣ 1️⃣ ｜7️⃣ 6️⃣
    狀況2總結：
  • 5️⃣ x0 < x2 < x1 < x3
  • 6️⃣ x2 < x0 < x3 < x1
  • 7️⃣ x2 < x0 < x1 < x3 (同 4️⃣)
  • 8️⃣ x0 < x2 < x3 < x1 (同 3️⃣)
    ▶️ x0 < x2 < x3 ↔️ x2 < x1 < x3
    ▶️ x2 < x0 < x3 ↔️ x0 < x3 < x1

狀況3(有重合的點)

  • x0 = x2 - x1 = x2
  • x0 = x3 - x1 = x3
    ⬇️整合
  • x2 <= x1 <= x3
  • x0 <= x2 <= x1
  • x0 <= x3 <= x1
